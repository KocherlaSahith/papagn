# Goodnight Animation 🌸

A cute pink-themed goodnight animation with floating hearts and stars!

## Deploy to Vercel

1. Push this project to a Git repository (GitHub, GitLab, or Bitbucket)
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Select your repository
5. Vercel will automatically detect this is a static HTML project
6. Click "Deploy"

The `vercel.json` configuration is already set up for automatic deployment!

## Features

- Soft pink gradient background with subtle animation
- Floating hearts with different pink shades
- Twinkling stars
- Rotating name display
- Floating heart animations
- Responsive design

## Usage

Just open `goodnight-animation.html` in a web browser to see the animation!
